# ROMs Directory

Place your Metal Slug 2 ROM file here.

## Required file:
- `mslug2.zip` - Metal Slug 2 ROM file

## Important Notes:
- The ROM file must be named exactly `mslug2.zip`
- Make sure it's a valid Neo Geo ROM dump
- The file should be in ZIP format
- Only place legally obtained ROM files here

## File Structure:
```
roms/
└── mslug2.zip  <-- Place your ROM file here
```

Once you place the ROM file here, you can start the web server and play the game!
